DESIGN SYSTEM SPECIFICATIONS
================================================================================

COLOR PALETTE
- Primary Cyan: #00d9ff (main accent color)
- Primary Light: #22d3ee
- Primary Dark: #0ea5e9
- Accent Purple: #a855f7
- Accent Pink: #ec4899
- Success: #10b981
- Warning: #f59e0b
- Error: #ef4444
- Info: #3b82f6
- Neutral Grayscale: Slate-50 through Slate-900
- Background: #000000 (pure black)
- Surface: #0f0f0f (cards, modals)

TYPOGRAPHY
- UI Font: IBM Plex Sans (modern, readable, professional)
- Code Font: JetBrains Mono (purpose-built for code)
- Type Scale: 5 levels (h1-h6 + body)
- Base Size: 16px (mobile) → 17px (desktop)
- Line Heights: 1.2 (headings), 1.6 (body), 1.75 (reading)

SPACING SYSTEM
- 8px base unit (all spacing derives from this)
- Scales: 4px, 8px, 16px, 24px, 32px, 48px, 64px, 96px
- Responsive scaling: 1.5x on tablet/desktop
- Grid system: Mobile first, breakpoints at 768px, 1024px, 1536px

SHADOWS & DEPTH
- 5 shadow levels (subtle to strong)
- Glow effects for cyan and purple accents
- Shadows enhance depth, never distract

BORDER RADIUS
- 6px (small elements, badges)
- 12px (cards, buttons)
- 16px (larger containers)
- 24px (hero sections)
- 9999px (pills, circular elements)

ANIMATIONS
- Fast: 150ms (small interactions)
- Base: 200ms (standard)
- Smooth: 300ms (larger elements)
- Slow: 500ms (full-page transitions)
- Timing: cubic-bezier(0.4, 0, 0.2, 1)
- Respects prefers-reduced-motion

STATE MANAGEMENT
================================================================================

Component States:
- Normal: Default appearance
- Hover: Color shift, lift (-2px), enhanced glow
- Focus: Cyan outline (2px), 4px offset
- Active: Deeper color, no lift
- Disabled: Reduced opacity (0.6), no interaction

Loading States:
- Skeleton screens: Gradient shimmer
- Progress bars: Smooth animation
- Spinners: Cyan rotating circle

Error States:
- Red border and text
- Icon with error symbol
- Clear message and recovery action

Success States:
- Green accent and icon
- Brief toast notification
- Auto-dismiss (3–4 seconds)

KEY DESIGN DECISIONS
================================================================================

1. Dark Theme (Always On)
   - Matches robotics/tech industry standards
   - Reduces eye strain for extended reading
   - Creates premium, sophisticated feel
   - Aligns with brand identity

2. Cyan as Primary Accent
   - Suggests advanced technology
   - High contrast on dark backgrounds
   - Represents precision and digital systems
   - Professional, not trendy

3. 8px Grid System
   - Professional standard
   - Ensures consistency
   - Simplifies responsive scaling
   - Predictable, maintainable

4. CSS Modules + Global Tokens
   - Prevents naming collisions
   - Easy component reuse
   - Custom properties for theming
   - Optimized bundle size

5. WCAG AAA (Not Just AA)
   - Sets highest accessibility standard
   - Ensures usability for all
   - Future-proofs against regulations
   - Demonstrates commitment to inclusion

6. Component-Based Architecture
   - Reusable across platform
   - Consistent behavior and styling
   - Easier maintenance and updates
   - Scales with team growth
